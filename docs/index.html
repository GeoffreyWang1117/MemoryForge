Quick Start

1

Clone & Install

git clone https://github.com/GeoffreyWang1117/MemoryForge.git
cd memoryforge
python -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
2

Configure Environment

cp .env.example .env
# Edit .env with your API keys
3

Start Services

# Start Qdrant and Neo4j
docker-compose up -d

# Start the API server
uvicorn memoryforge.api.app:app --reload
4

Use the API

import asyncio
from memoryforge.memory.working.memory import WorkingMemory
from memoryforge.core.types import MemoryEntry, MemoryLayer, ImportanceScore

async def main():
    wm = WorkingMemory(max_entries=100)

    entry = MemoryEntry(
        content="User wants to build a REST API",
        layer=MemoryLayer.WORKING,
        importance=ImportanceScore(base_score=0.8),
    )
    await wm.store(entry)

asyncio.run(main())

REST API

Method Endpoint Description
GET /health Health check endpoint
POST /api/v1/memory/store Store a new memory entry
POST /api/v1/memory/query Query memories by content
GET /api/v1/memory/list List all stored memories
DELETE /api/v1/memory/{id} Delete a memory by ID
GET /api/v1/memory/stats Get memory statistics
POST /api/v1/persistence/save Persist memories to SQLite
GET /api/v1/persistence/load Load memories from SQLite

Example: Store a Memory

curl -X POST http://localhost:8000/api/v1/memory/store \
  -H "Content-Type: application/json" \
  -d '{
    "content": "User prefers Python for backend development",
    "importance": 0.8,
    "tags": ["preference", "python", "backend"]
  }'
